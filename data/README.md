# Sample Data

This directory contains sample trials for testing the pipeline.

## Contents

- `sample_a/` — One trial from Dataset A (7-DoF, `label.json` format)
- `sample_c/` — First 10 frames from Dataset C (6-DoF, numbered `.txt` format)

## Organizing Your Own Data

To train on the full datasets, organize your data under a root directory:

```
data_root/
├── 7DOF2024/                      # Dataset A
│   ├── Attempt 1 - .../Trial1/
│   │   ├── label.json
│   │   ├── Frames/                # Video frames (for detection training)
│   │   ├── YOLO_FEATURES/         # Precomputed (generated by pipeline)
│   │   ├── DEPTH/embeddings/      # Precomputed (generated by pipeline)
│   │   ├── POSE_FEATURES/         # Precomputed (generated by pipeline)
│   │   └── _visual_cache.npz      # Precomputed (generated by pipeline)
│   ├── .../Trial2/
│   └── Trial33/                   # Trials 33+ at top level
│
├── BAPES2024/                     # Dataset B
│   ├── Trial1/
│   └── ...
│
└── 6DOF2023/                      # Dataset C
    ├── Test 1/
    └── ...
```

Then edit `configs/paths.yaml`:

```yaml
data_root: "/path/to/data_root"
dataset_a: "7DOF2024"
dataset_b: "BAPES2024"
dataset_c: "6DOF2023"
```

See [`docs/DATA_FORMAT.md`](../docs/DATA_FORMAT.md) for detailed format specifications.
